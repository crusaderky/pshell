Installation
============

Required dependencies
---------------------

- Python 3.8 or later
- `psutil <https://github.com/giampaolo/psutil>`_ 5.7 or later


Installing with conda
---------------------

.. code-block:: bash

    conda install -c conda-forge pshell


Installing with pip
-------------------

.. code-block:: bash

    pip install pshell


.. _mindeps_policy:

Minimum dependency versions
---------------------------

This project adopts a rolling policy based on `SPEC 0
<https://scientific-python.org/specs/spec-0000/>`_ regarding the minimum
supported version of its dependencies.

You can see the actual minimum tested versions in `pyproject.toml
<https://github.com/crusaderky/pshell/blob/main/pyproject.toml>`_.
